**Prerequisites**

1. Node - 7.5.0
2. Protractor - 5.1
3. Bower - 1.8
4. Grunt - 1.0.1

Steps to run the app:-

1. ``npm install``
2. ``bower install``
2. ``grunt``


Steps to run the e2e tests:-

1. ``protractor e2e/conf.js``


You can see the project in action over here - https://tinker20.github.io/spotify-music-search/#!/music-search
